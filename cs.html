1. AM Modulation

Real-Life Example:
 Amplitude Modulation (AM) is commonly used in AM radio broadcasting, where audio signals are transmitted over long distances by modulating a carrier wave. It is also used in early aviation communications.

Simulation of Amplitude Modulation (AM) using MATLAB:

        % MATLAB Code for AM Modulation
        fs = 1000; % Sampling frequency
        t = 0:1/fs:1-1/fs; % Time vector
        carrier = cos(2*pi*100*t); % Carrier signal
        message = cos(2*pi*5*t); % Message signal
        am_signal = (1 + message) .* carrier; % AM modulation
        plot(t, am_signal);
        title('AM Modulated Signal');

2. FM Modulation

Description:
 Frequency Modulation (FM) is a method of modulating a carrier signal where the instantaneous frequency of the carrier varies in proportion to the amplitude of the modulating signal. FM is widely used in communication systems due to its resilience to noise and improved signal quality.

Simulation of Frequency Modulation (FM) using MATLAB:

% MATLAB Code for FM Modulation
fs = 1000; % Sampling frequency
t = 0:1/fs:1-1/fs; % Time vector
carrier = cos(2*pi*100*t); % Carrier signal
message = cos(2*pi*5*t); % Message signal
fm_signal = cos(2*pi*100*t + sin(2*pi*5*t)); % FM modulation
plot(t, fm_signal);
title('FM Modulated Signal');

3. PCM

Description:
 PCM is a digital modulation technique where an analog signal is sampled at regular intervals, quantized to the nearest level within a range, and encoded into binary form. The process involves:
● Sampling: The continuous-time signal is sampled at a specific frequency (e.g., 4kHz or 8kHz) according to the Nyquist criterion.
● Quantization: The amplitude of the sampled signal is approximated to the nearest quantization level.
● Encoding: The quantized levels are converted into binary codes for digital transmission or storage.

Simulation of PCM using MATLAB:

        % MATLAB Code for PCM Modulation
        % PCM Implementation in MATLAB
        
        % Parameters
        fs = 8000; % Sampling frequency (Hz)
        t = 0:1/fs:1-1/fs; % Time vector (1 second duration)
        f = 5; % Frequency of the signal (Hz)
        
        % Generate a sine wave signal
        x = sin(2 * pi * f * t);
        
        % Quantization parameters
        numLevels = 16; % Number of quantization levels
        x_min = -1; % Minimum amplitude of the signal
        x_max = 1; % Maximum amplitude of the signal
        
        % Quantization step size
        delta = (x_max - x_min) / numLevels;
        
        % Quantize the signal
        quantized_signal = round((x - x_min) / delta) * delta + x_min;
        
        % Encode the quantized signal
        encoded_signal = round((quantized_signal - x_min) / delta);
        
        % Decode the quantized signal (Reconstruct)
        decoded_signal = encoded_signal * delta + x_min;
        
        % Plot results
        figure;
        subplot(3,1,1);
        plot(t, x);
        title('Original Signal');
        xlabel('Time (s)');
        ylabel('Amplitude');
        
        subplot(3,1,2);
        stem(t, encoded_signal, 'filled');
        title('PCM Encoded Signal');
        xlabel('Time (s)');
        ylabel('Quantization Levels');
        
        subplot(3,1,3);
        plot(t, decoded_signal);
        title('PCM Decoded Signal');
        xlabel('Time (s)');
        ylabel('Amplitude');

4. DPCM

Description:
● DPCM is an enhanced version of PCM that reduces redundancy by encoding the difference between consecutive samples rather than the absolute sample values.
● Encoding: The predictor estimates the current sample using previous samples, and the difference (error) is quantized and encoded.
● Decoding: The difference is added back to the predictor value to reconstruct the original signal.

Simulation of DPCM using MATLAB:

% MATLAB Code for DPCM Modulation
fs = 8000; % Sampling frequency (Hz)
t = 0:1/fs:1-1/fs; % Time vector (1 second duration)
f = 5; % Frequency of the signal (Hz)

% Generate a sine wave signal
x = sin(2 * pi * f * t);

% Quantization levels
numLevels = 16; % Number of quantization levels
delta = 2 / numLevels; % Step size for quantization

% Initialize variables
n = length(x);
encoded = zeros(1, n);
decoded = zeros(1, n);
prediction = 0; % Initial prediction

% DPCM encoding and decoding
for i = 1:n
    % Calculate difference
    diff = x(i) - prediction;
    
    % Quantize the difference
    quantized_diff = round(diff / delta) * delta;
    
    % Store encoded signal
    encoded(i) = quantized_diff / delta; % Encoded as indices
    
    % Reconstruct the signal
    decoded(i) = prediction + quantized_diff;
    
    % Update the prediction (use reconstructed value)
    prediction = decoded(i);
end

% Plot the results
figure;
subplot(3,1,1);
plot(t, x);
title('Original Signal');
xlabel('Time (s)');
ylabel('Amplitude');

subplot(3,1,2);
stem(t, encoded, 'filled');
title('DPCM Encoded Signal');
xlabel('Time (s)');
ylabel('Encoded Value');

subplot(3,1,3);
plot(t, decoded);
title('DPCM Decoded Signal');
xlabel('Time (s)');
ylabel('Amplitude');

6.DSBSC

Real-Life Example:
 Double Sideband Suppressed Carrier (DSB-SC) is a type of amplitude modulation (AM) where the carrier component is suppressed, leaving only the sidebands in the transmitted signal. This improves power efficiency, as no energy is spent on transmitting the carrier.

7.ASK

Real-Life Example:
 ● ASK is a digital modulation technique where the amplitude of the carrier signal is varied in accordance with the binary data (0s and 1s) of the modulating signal.
● Binary '1': The carrier signal is transmitted at its original amplitude.
● Binary '0': The carrier signal is turned off or transmitted with zero/minimal amplitude.
● Mathematically, the ASK signal is represented as: s(t)=
Ac*cos*(2πfct), for binary “1”.
s(t)=0 , for binary “0”.

Simulation of ASK using MATLAB:

    % MATLAB Code for ASK
    % Amplitude Shift Keying (ASK) Modulation and Demodulation
clc;
clear;
close all;

% Parameters
fs = 1000; % Sampling frequency in Hz
fc = 100;  % Carrier frequency in Hz
bit_rate = 10; % Bit rate in bits per second
bits = [1 0 1 1 0 0 1]; % Data bits to transmit

% Derived parameters
T = 1/bit_rate; % Bit duration in seconds
N = fs * T; % Number of samples per bit
t = 0:1/fs:T-1/fs; % Time vector for one bit
carrier = cos(2 * pi * fc * t); % Carrier signal

% ASK Modulation
modulated_signal = [];
for bit = bits
    if bit == 1
        modulated_signal = [modulated_signal carrier]; % Transmit carrier for bit 1
    else
        modulated_signal = [modulated_signal zeros(1, length(t))]; % Transmit zero for bit 0
    end
end

% Time vector for modulated signal
total_time = length(bits) * T;
t_modulated = 0:1/fs:total_time-1/fs;

% Plot modulated signal
figure;
subplot(2,1,1);
stem(bits, 'filled');
title('Input Data Bits');
ylabel('Amplitude');
axis([0 length(bits)+1 -0.5 1.5]);
grid on;

subplot(2,1,2);
plot(t_modulated, modulated_signal);
title('ASK Modulated Signal');
ylabel('Amplitude');
xlabel('Time (s)');
grid on;

8.FSK

Real-Life Example:
●Frequency Shift Keying (FSK) is a digital modulation technique where the frequency of the carrier signal is varied based on the binary message signal.
● Binary Representation:
o Binary '1' is transmitted using a high-frequency carrier (f1).
o Binary '0' is transmitted using a low-frequency carrier (f2).
● Modulation Process: The binary message signal modulates the carrier frequency. For m(t)=1, the carrier signal is s(t)=A*cos*(2πf1t) and for m(t)=0, it is s(t)=A*cos*(2πf2t).
● Demodulation Process: The received FSK signal is demodulated by separating the two frequencies using band-pass filters tuned to f1 and f2. The signal power is measured in each band, and a decision threshold determines the binary output.

Simulation of FSK using MATLAB:

    % MATLAB Code for FSK
    
    % Parameters
fs = 1e5;          % Sampling frequency (Hz)
Tb = 1e-3;         % Bit duration (s)
Rb = 1 / Tb;       % Bit rate (bps)
f0 = 1e3;          % Frequency for bit '0' (Hz)
f1 = 2e3;          % Frequency for bit '1' (Hz)
numBits = 10;      % Number of bits in the message

% Generate random binary data
data = randi([0 1], 1, numBits);

% Time vector for one bit
t = 0:1/fs:Tb-1/fs;

% Modulation process
FSK_signal = [];
for i = 1:numBits
    if data(i) == 0
        carrier = cos(2 * pi * f0 * t); % Frequency for bit '0'
    else
        carrier = cos(2 * pi * f1 * t); % Frequency for bit '1'
    end
    FSK_signal = [FSK_signal carrier];
end

% Time vector for entire signal
time = 0:1/fs:numBits*Tb-1/fs;

% Plotting the results
figure;

subplot(2,1,1);
stairs(0:numBits-1, data, 'LineWidth', 2);
grid on;
xlabel('Bit Index');
ylabel('Data');
title('Input Binary Data');
ylim([-0.5 1.5]);

subplot(2,1,2);
plot(time, FSK_signal, 'LineWidth', 1.5);
grid on;
xlabel('Time (s)');
ylabel('Amplitude');
title('FSK Modulated Signal');

9.PSK

Real-Life Example:
 ●Phase Shift Keying (PSK) is a digital modulation technique where the phase of the carrier signal is varied based on the binary message signal.
● Binary Representation:
o Binary '1' is transmitted with a carrier signal having a phase of 0∘0^\circ0∘.
o Binary '0' is transmitted with a carrier signal having a phase of 180∘180^\circ180∘.
● Modulation Process: The binary message signal modulates the phase of the carrier signal. For m(t)=1, the modulated signal is s(t)=Ac*cos*(2πfct) and m(t)=0, it is s(t)=−Ac*cos*(2πfct).
● Demodulation Process: The received PSK signal is demodulated by correlating it with a reference carrier signal. The resulting signal is integrated over the bit duration. A decision threshold is applied:
o Positive values correspond to binary '1'.
o Negative values correspond to binary '0'.

Simulation of PSK using MATLAB:

    % MATLAB Code for PSK
    
    % Parameters
fs = 1e5;          % Sampling frequency (Hz)
Tb = 1e-3;         % Bit duration (s)
Rb = 1 / Tb;       % Bit rate (bps)
fc = 1e3;          % Carrier frequency (Hz)
numBits = 10;      % Number of bits in the message

% Generate random binary data
data = randi([0 1], 1, numBits);

% Time vector for one bit
t = 0:1/fs:Tb-1/fs;

% BPSK Modulation Process
BPSK_signal = [];
for i = 1:numBits
    if data(i) == 0
        carrier = cos(2 * pi * fc * t + pi); % Phase shifted by 180° for bit '0'
    else
        carrier = cos(2 * pi * fc * t);      % No phase shift for bit '1'
    end
    BPSK_signal = [BPSK_signal carrier];
end

% Time vector for the entire signal
time = 0:1/fs:numBits*Tb-1/fs;

% Plotting the Results
figure;

subplot(2,1,1);
stairs(0:numBits-1, data, 'LineWidth', 1.5);
grid on;
xlabel('Bit Index');
ylabel('Data');
title('Input Binary Data');
ylim([-0.5 1.5]);

subplot(2,1,2);
plot(time, BPSK_signal, 'LineWidth', 1.5);
grid on;
xlabel('Time (s)');
ylabel('Amplitude');
title('BPSK Modulated Signal');
